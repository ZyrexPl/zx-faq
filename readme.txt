Frequently asked questions.

== Description ==
Plugin allows you to create and add to the page frequently asked questions.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through `Plugins` screen in WordPress Admin Panel.
